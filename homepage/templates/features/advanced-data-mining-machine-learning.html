Advanced Data Mining and Machine Learning Features

Orange is a state-of-the-art open-source data mining software. It comes with a user-friendly UI, while at its core is a scripting, Python-based library for fast prototyping of new machine learning algorithms. The program is packed with a large variety of standard and not-so-standard tools that can be supplemented with specialized add-ons adapted for data analysis in different domains.

Powerful Machine Learning Library

We rely on powerful Python libraries such as NumPy, SciPy, and scikit-learn, which allow for a quick and easy integration with other advanced data mining tools. Regardless of whether you are using Orange through scripting or GUI, you will find it contains all common supervised machine learning algorithms, such as classification trees, support vector machines, linear and logistic regression, random forest, naïve bayes, nearest neighbors, CN2; and unsupervised machine learning algorithms, such as k-means and hierarchical clustering tools for feature subset selection and rapid cross-validation of trained models, confusion matrix, model scoring, and much, much more! {% comment %}Content goes here. Big Data

Load data from large databases using SQL widget. Even if you have a database the size of several gigabytes, Orange will use sampling to quickly deliver results. Queries can be easily passed to other widgets for visualization, prediction and scoring. Install 2UDA suite to access all the great functionalities for big data analytics.

Specialized Add-Ons

Add-ons provide domain-specific techniques that can be used on top of the core Orange distribution. Dive into text analysis with Text Mining, uncover and explore graphs with Networks, merge different data sets with Data Fusion, discover new gene candidates with Bioinformatics, and infer association rules with Associate add-on.
