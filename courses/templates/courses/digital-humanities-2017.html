Hands-on Text Analytics with Orange

Dear workshop participants,

Welcome to the Hands-on Text Analytics with Orange! To save us precious time, we kindly ask you to install Orange software before the workshop.

Installation Instructions

Please visit our website and download the installation package for your operating system. If you are a Linux user or wish to install the software via command line, we warmly suggest you use the Anaconda framework. Installation instructions are described on our website and on our Github page.

Please, open the software and check that everything works before the workshop. To get you started, here are some introductory YouTube tutorials.

You must also install two add-ons for the workshop: Text and Image Analytics. In Orange go to Options → Add-ons and select Text and Image Analytics. Once the installation is complete, restart Orange. Alternatively, follow video tutorial for instructions.

Should you experience any problems with your installation, we will be available an hour before the workshop. You can catch us in the lobby and we’ll help you out.

We're looking forwards to seeing you at the workshop!


Ajda, Niko and Lan
